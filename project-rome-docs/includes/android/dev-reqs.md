Connected Devices のすべての機能には、Android アプリ開発 IDE と、サポートされているアーキテクチャ (**armeabi-v7a**、**arm64-v8a**、**x86**、または **x86_64**) のいずれかを搭載した Android デバイス、またはエミュレーターが必要です。 システムで Android 4.4.2 またはそれ以降を実行している必要があります。
